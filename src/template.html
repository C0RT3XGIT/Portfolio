Memory Game

Main idea is that order of the images will be randomized and faced down. You will have to find pair of image that you have chosen from upper row. If the images will be the same, there will be displayed a pop-up alert that says “Congratulations!” else the shown alert will display “Try again !” and user will have to try once again to find pair of selected picture.
